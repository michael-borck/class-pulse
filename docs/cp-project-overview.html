Final: Launch & Usage Guide

Project Overview

ClassPulse is a minimal web-based audience interaction tool for the classroom or events, inspired by Mentimeter. Presenters can create real-time polls (multiple choice, word cloud, rating scale), and participants submit responses using a session code or QR code. Results are visualized live for both presenters and the audience.

  • Simple and clean UI (Tailwind CSS)
  • Bar, word cloud, and scale visualizations (Chart.js/ECharts)
  • Real-time updates (WebSocket, Socket.IO)
  • Persistent & exportable results (SQLite, CSV)
  • Secure joining with session code or QR code

Technology Stack

Flask (Backend)
SQLite (Persistence)
Flask-SocketIO (Realtime)
Tailwind CSS
Chart.js / ECharts
Vanilla JS, Font Awesome

Quickstart: Setup & Run

  1. Clone/Create the project directory
    mkdir -p ~/projects/classpulse && cd ~/projects/classpulse
  2. Install Python dependencies
    cat > requirements.txt <<EOF Flask==2.2.5 flask-socketio==5.3.6 eventlet==0.35.1 Flask-Login==0.6.3 Flask-Cors==4.0.0 SQLAlchemy==2.0.28 qrcode==7.4.2 EOF python3 -m venv venv source venv/bin/activate pip install -r requirements.txt
  3. Configure and launch the app
    # Run the Flask-SocketIO server with eventlet export FLASK_APP=app.py export FLASK_ENV=development flask run --host=0.0.0.0
  4. Access in browser
    http://localhost:5000/

For production, see Flask-SocketIO deployment docs.

Core Features: At a Glance

  1. Presenter Login: Hardcoded/simple login for presenters.
  2. Create & Manage Questions: Add multiple-choice, rating, and word clouds. Control which question is live.
  3. Session Join for Audience: Participants join by session URL/code or QR code. Each session has a unique identifier.
  4. Real-time Updates: Submissions instantly update results on the presenter dashboard and the audience display via WebSocket.
  5. Visualization:
    • Multiple-choice: Bar charts (Chart.js)
    • Word cloud: (ECharts)
    • Scale: Horizontal bar (Chart.js)
  6. Export & Persist: Download all session results as CSV. Results are not lost on reload (SQLite or JSON-based persistence).
  7. Theming: Toggle simple light/dark mode on the dashboard (Tailwind utility).

Joining a Session: QR/Code

To join, an audience member can scan a QR code from their device, or enter the session code at the provided link. This QR code is dynamically generated for each session using qrcode Python library or a frontend library.

Exporting Results

All session results can be exported as a CSV file by the presenter – whether for grades, participation, or further analysis. This is accessible from the presenter dashboard.
