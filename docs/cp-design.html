Overview and Features

ClassPulse is an open-source, web-based audience interaction tool inspired by platforms like Mentimeter. It enables presenters to create and manage live questions for interactive sessions, while audience members can join via a session code or QR code to submit responses. All results are visualized in real-time, making it perfect for classrooms, workshops, or any group engagement.

  • Presenter dashboard to create and manage questions
  • Audience joins using unique code or QR
  • Real-time results with live visualizations
  • Multiple question types: Multiple Choice, Word Cloud, Rating Scale

Bonus Features (optional):

  • Export results to CSV
  • Session persistence
  • Light/Dark mode

Technology Stack & Libraries

Backend

  • Python Flask: Web application framework
  • Flask-SocketIO: Real-time events over WebSockets
  • SQLite3: Persistent but lightweight database

Frontend

  • Tailwind CSS (via CDN): Utility-first, fast web styling (v2.2.19)
  • Chart.js: Bar/Scale visualizations (v4.4.3)
  • Font Awesome: Vector icons (v6.5.2)
  • Google Fonts: Elegant typography
  • Vanilla JS: Dynamic interactivity

Backend Implementation

  • Session Management: Each session gets a unique join code.
  • Question CRUD: Presenters create, read, update, and delete questions.
  • WebSocket Endpoints: Real-time updates for question/response events.
  • Database: Store sessions, questions, responses, and optionally presenter logins.

Real-Time Communication

Socket.IO (for Flask): Enables pushing live updates—no page reloads required.
Presenters see live chart/word cloud updates as soon as responses submit. 

/* audience.js (Client-Side) */

socket.emit('join', { session_code: 'ABC123' });

function submitAnswer(answer) {
  socket.emit('submit_response', {
    session_code: 'ABC123',
    answer: answer
